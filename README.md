# El Pollo Loco

> A 2D side-scrolling platformer game built with HTML5 Canvas and vanilla JavaScript — training project from Developer Akademie.

## About

El Pollo Loco is a browser-based 2D platformer where you control a character named Pepe navigating through a desert landscape. Fight chickens, collect coins and bottles, and defeat the endboss. The project demonstrates object-oriented JavaScript, HTML5 Canvas rendering, sprite animation, and responsive game design with mobile touch controls.

## Tech Stack

- HTML5 Canvas API
- CSS3 (custom properties, media queries, responsive design)
- Vanilla JavaScript (ES6 classes)
- No frameworks or build tools — pure static files

## Features

- Side-scrolling platformer with parallax backgrounds
- Keyboard controls (arrow keys, WASD, space, E) and mobile touch controls
- Enemy AI (chickens, poults, endboss)
- Collectables (coins for healing, bottles for throwing)
- Health and status bar system
- Sound effects and background music with mute toggle
- Fullscreen mode
- Framerate settings (30–360 FPS)
- Responsive layout with landscape lock on mobile devices

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optionally: a local development server (e.g., VS Code Live Server extension)

### Running the Game

1. Clone the repository:
   ```bash
   git clone https://github.com/SimonHeistermann/El-Pollo-Loco.git
   ```
2. Open `index.html` in your browser, or serve the directory with a local server:
   ```bash
   # Example with Python
   python3 -m http.server 8000
   ```
3. Click **Start Game** to play.
